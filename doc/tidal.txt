==============================================================================
1. INTRODUCTION                                                    *tidal-intro*

tidal.nvim provides two features:
  - A floating terminal that toggles over your editor
  - A Claude AI split terminal with a Telescope session history picker

==============================================================================
2. REQUIREMENTS                                            *tidal-requirements*

  - Neovim 0.10+
  - telescope.nvim (https://github.com/nvim-telescope/telescope.nvim)
  - Claude CLI (https://claude.ai/code) — for the Claude split only

==============================================================================
3. SETUP                                                          *tidal-setup*

Call `require("tidal").setup(opts)` with your lazy.nvim spec or init.lua.
Calling setup with no arguments applies all defaults.

Example (lazy.nvim): >lua
  {
    "Vasilis-Narain/tidal.nvim",
    dependencies = { "nvim-telescope/telescope.nvim" },
    config = function()
      require("tidal").setup()
    end,
  }
<

==============================================================================
4. COMMANDS                                                     *tidal-commands*

:Tidal                  Toggle the floating terminal.
:TidalClaude            Toggle the Claude AI side split.
:TidalPick              Open the Telescope Claude session picker.

==============================================================================
5. KEYMAPS                                                       *tidal-keymaps*

Default keymaps (set in normal mode by `setup()`):

  <leader>tt    Toggle floating terminal
  <leader>tc    Toggle Claude split (narrow, 33%)
  <leader>tC    Toggle Claude split (wide, 50%)
  <leader>tlc   Open Claude session picker

Inside the Claude split (terminal mode):

  <Esc>         Leave terminal mode (standard <C-\><C-n>; not configurable)
  <C-q>         Send <Esc> to Claude — cancel prompts or dismiss menus
                without leaving terminal mode

Configure <C-q> via `claude.escape` — set to another lhs to rebind, or
`false`/`""` to disable: >lua
  require("tidal").setup({
    claude = { escape = "<C-g>" },
  })
<

Inside the session picker (`:TidalPick` / `<leader>tlc`):

  <M-d>         Delete chat under cursor — removes the .jsonl from disk
                and refreshes the picker
  <C-d>         Move highlighted chat down one slot
  <C-u>         Move highlighted chat up one slot

Chat order persists across Neovim sessions in `.tidal-order.json` next to
the chat files. Configure via `sessions.delete_key`, `sessions.move_up_key`,
`sessions.move_down_key`. Set a key to `false`/`""` to disable — disabling
the move keys restores Telescope's default preview-scroll mappings on
<C-u>/<C-d>: >lua
  require("tidal").setup({
    sessions = {
      delete_key    = "<C-x>",
      move_up_key   = false,
      move_down_key = false,
    },
  })
<

Disable all global keymaps: >lua
  require("tidal").setup({ keymaps = {} })
<
Disable a specific keymap: >lua
  require("tidal").setup({ keymaps = { toggle_terminal = false } })
<

See also KEYBINDS.md in the repo root for the full reference table.

6. HIGHLIGHTS                                                 *tidal-highlights*

  TidalUser        User message prefix in previewer
  TidalUserBody    User message body in previewer
  TidalAssistant   Assistant header in previewer
  TidalBody        Assistant body text in previewer

Defaults link to standard groups: Title, Comment, Special, Normal.

Override example: >lua
  require("tidal").setup({
    highlights = {
      user      = { fg = "#7aa2f7", bold = true },
      assistant = { fg = "#e0af68", bold = true },
    },
  })

==============================================================================
7. CONFIGURATION                                                  *tidal-config*

Full defaults: >lua
  require("tidal").setup({
    terminal = {
      shell  = nil,       -- nil = auto-detect (Git bash on Windows, $SHELL elsewhere)
      width  = 0.8,       -- fraction of screen width
      height = 0.8,       -- fraction of screen height
      border = "rounded",
    },
    claude = {
      cmd        = "claude",   -- path to Claude CLI binary
      fraction   = 0.33,       -- default split width fraction
      statusline = " Claude",
      escape     = "<C-q>",    -- terminal-mode key sending <Esc> to Claude
    },
    sessions = {
      max           = 20,           -- max sessions shown in picker
      max_bytes     = 262144,       -- max bytes read from JSONL for preview
      delete_key    = "<M-d>",      -- picker: delete chat under cursor
      move_up_key   = "<C-d>",      -- picker: move chat up
      move_down_key = "<C-u>",      -- picker: move chat down
    },
    highlights = {
      user      = { fg = "#7aa2f7", bold = true },
      user_body = { fg = "#bb9af7", italic = true },
      assistant = { fg = "#e0af68", bold = true },
      body      = { fg = "#c0caf5" },
    },
    keymaps = {
      toggle_terminal   = "<leader>tt",
      toggle_claude     = "<leader>tc",
      toggle_claude_big = "<leader>tC",
      claude_pick       = "<leader>tlc",
    },
  })
<

==============================================================================
8. SECURITY                                                    *tidal-security*

  - `claude.cmd` is executed as argv[0] via |jobstart()| (resolved through
    $PATH). Only set it to a binary you trust — a malicious value pasted
    from an untrusted config runs arbitrary code on toggle.
  - `terminal.shell` is executed the same way. Same trust boundary applies.
  - The session picker reads and deletes .jsonl files under
    `~/.claude/projects/<encoded-cwd>/`. Delete rejects symlinks and paths
    outside the project dir; the resume flow rejects session ids that are
    not UUID-shaped.
